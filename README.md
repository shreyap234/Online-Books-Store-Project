📌 Project Title: Online Book Store Database

📖 Description:

This project is based on an Online Book Store Management System where I created and analyzed a database using SQL.

The main goal of this project is to manage book records, customer details, and order transactions, and to extract useful insights using SQL queries.

🛠️ Tools & Technologies:
SQL (MySQL)
Database Concepts (Tables, Joins, Aggregations)

📂 Database Structure:

The project contains 3 main tables:

Books – stores book details (title, author, genre, price, stock, published year)

Customers – stores customer information (name, city, etc.)

Orders – stores order details (book_id, customer_id, quantity, total amount, date)

🔍 Key Features / Queries:

Retrieve books by genre and published year

Find total stock and most expensive book 

Analyze customer orders and spending

Calculate total revenue from orders

Find most frequently ordered book

Get top-selling genres and authors

Identify customers with highest purchases

Use JOIN, GROUP BY, HAVING, and aggregate functions

📊 Learning Outcomes:
Strong understanding of SQL queries

Practice of JOIN operations

Use of aggregate functions (SUM, AVG, COUNT)

Working with real-world database scenarios

Data analysis and insight generation

🎯 Conclusion:
This project helped me improve my SQL skills and understand how databases are used in real-world applications like e-commerce platforms.
